# Student Record Management System

## Problem Statement
Many schools, tutors, and small training centers need a simple way to manage student information. Manual record handling can be slow and confusing, especially when searching, updating, and deleting details.

## Objective
Build a basic Python command-line application to manage student records in an organized way.

## Features
- Add a student
- View all students
- Search for a student
- Update student details
- Delete a student
- Save records to a file

## Technologies Used
- Python 3

## How to Run
1. Open terminal in the project folder.
2. Run: python main.py
